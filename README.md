# Algorithms and Data Structures

This repository contains implementations of core algorithms and data structures including trees, graphs, sorting, heaps, and a Turing machine. It is designed for learning, visualization, and performance testing.

## Projects

- [AVL and Binary Search Trees](./avl_and_binary_trees)  
  Implementation and comparison of BST and AVL trees with performance graphs and unit tests.

- [Heap](./heap)  
  Max-heap and min-heap implementations with priority queue examples.

- [Sorting Algorithms](./sorting)  
  Classic sorting algorithms with execution time analysis.

- [Turing Machine](./turing)  
  Turing machine simulation with configurable transition rules.

## Features

- Python scripts for generating performance graphs
- Visual aids to better understand algorithmic behavior
- Clear separation of logic and testing
